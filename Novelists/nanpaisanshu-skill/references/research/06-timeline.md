# 南派三叔（徐磊）完整时间线

> 来源：百度百科、公开采访、媒体报道综合整理

---

## 时间线总表

| 时间 | 事件 | 意义 |
|------|------|------|
| **1982年2月20日** | 出生于浙江省嘉兴市嘉善县 | 浙江水乡成长背景，为日后创作提供地域文化养分 |
| **1987年（5岁）** | 在外婆怀中听到第一个与尸体有关的故事 | 童年听到"血尸"故事，对地下神秘世界产生浓厚想象，成为创作原点 |
| **1990年代** | 就读于嘉善二中 | 中学时期开始尝试写作，以身边人为原型构建想象中的世界 |
| **2000-2004年** | 就读于浙江树人大学 | 大学期间继续积累创作素材 |
| **2004-2006年** | 从事广告美工、软件编程、国际贸易等行业 | 多元职业经历为创作提供社会阅历和商业思维基础 |
| **2006年6月26日** | 以ID"218.109.112.*"在贴吧发布《七星鲁王宫》，开始网络文学创作 | 《盗墓笔记》系列起点，从贴吧起步获得首批书粉 |
| **2006年下半年** | 连载地从贴吧转移到起点中文网 | 进入专业网文平台，读者群体扩大 |
| **2007年1月** | 《盗墓笔记：七星鲁王宫》实体书出版，首月销量突破60万册 | 正式进军出版界，开启畅销书作家之路 |
| **2007年4月** | 《盗墓笔记贰：秦岭神树》出版 | 系列化运作初现雏形 |
| **2007年11月** | 《盗墓笔记叁：云顶天宫》上市 | 世界观持续扩展 |
| **2008年11月** | 《盗墓笔记肆：蛇沼鬼城》出版 | 系列进入中期发展阶段 |
| **2009年7月** | 《盗墓笔记伍：谜海归巢》出版，第一季完结 | 阶段性收官，为第二季铺垫 |
| **2009年12月** | 《盗墓笔记陆：阴山古楼》出版，第二季开启 | 开启新篇章，延续系列热度 |
| **2010年4月** | 《大漠苍狼：绝地勘探》出版 | 尝试盗墓题材之外的探险小说 |
| **2010年9月** | 《盗墓笔记柒：邛笼石影》出版 | 系列持续推进 |
| **2010年11月** | 《怒江之战》系列完结并上市 | 二战军事探险题材拓展 |
| **2010年** | 以285万元版税荣登"第五届中国作家富豪榜"第14位 | 首次进入作家富豪榜，商业认可度提升 |
| **2011年** | 担任磨铁图书副总编辑，创办杂志《超好看》并担任主编 | 从作家向出版人转型，首印30万册，上市第二天卖断货 |
| **2011年9月7日** | "南派三叔漫工厂"工作室正式成立 | 进军漫画领域，与沧月、颜开、姚非拉等合作 |
| **2011年11月** | 以1580万元版税荣登"第六届中国作家富豪榜"第2位 | 商业巅峰期，成为顶级畅销书作家 |
| **2011年12月19日** | 《盗墓笔记捌：大结局（上下）》出版，首周销量破100万册 | 系列正式完结，九部总销量超1200万册，奠定代表作地位 |
| **2012年8月** | 《藏海花》出版，讲述张起灵身世之谜 | 开启后传系列，延续IP生命力 |
| **2012年** | 以850万元版税荣登"第七届中国作家富豪榜"第9位 | 持续保持高收入作家地位 |
| **2012年10月** | 《漫绘SHOCK》停刊，新刊《惊叹号》诞生 | 杂志业务调整 |
| **2013年2月** | 《沙海：荒沙诡影》出版 | 《盗墓笔记》少年篇启动 |
| **2013年3月22日** | 微博宣布因压力巨大封笔，不再进行文学创作 | 创作生涯重大转折，引发广泛关注与讨论 |
| **2013年8月** | 《沙海：沙蟒蛇巢》出版 | 封笔前最后出版作品 |

## 阶段划分

### 第一阶段：成长与积累（1982-2006）
- 童年故事启蒙 → 中学写作尝试 → 大学积累 → 多元职业经历

### 第二阶段：创作崛起（2006-2011）
- 网络连载起步 → 实体书出版 → 系列化运作 → 作家富豪榜前列

### 第三阶段：商业扩张（2011-2013）
- 创办杂志 → 成立工作室 → 系列完结 → 封笔

### 第四阶段：IP运营转型（2014-至今）
- 复出 → 成立影视公司 → 融资扩张 → 全面IP生态运营 → 主流认可
