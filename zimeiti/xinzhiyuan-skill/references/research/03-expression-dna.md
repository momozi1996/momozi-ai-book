# 03 — 表达DNA与风格

## 核心风格：中文叙事 + 惊叹号密集

### 五大语言DNA

1. **中文语境叙事**
   - 优先用中文语境叙事，鲜见直接英文长句
   - 英文术语常变「中文语境化」

2. **故事新闻体——先冲突再技术**
   - 每篇开头先设一个悬念或冲突，再展开技术细节
   - 「9秒，一家公司没了！Claude删库跑路……」
   - 「Anthropic封杀110人公司，API还在照常计费……」

3. **惊叹号密度极高**
   - 标题+正文每段至少1.5个感叹号
   - 「炸！冲！崩了！反杀！封神！洗牌！」
   - 制造强烈紧迫感和「先读」

4. **多路径叙事结构**
   - 宏观→技术→微细→全局→最后定「意识形态回顾」
   - 「背景 → 数据 → 技术细节 → 全球格局 → 评论」
   - AIGCRank榜单标准格式

5. **「编辑名单」署名制**
   - 全程署名：新智元报道 + 编辑花名
   - 技术角度：「元宇」，讲故事：「元E/其它花名」

### 招牌句式

**长文叙事式：**
```
[悬念/冲突标题]
[背景铺垫] → [细节递进] → [技术拆解] → [全局格局] → [结论信号]
```

**新闻短讯体：**
```
[标题关键词+惊叹号]
一句话描述 → 更多细节 → 外部链接
——新智元 · AIEra
```

**追踪型句式：**
```
[原标题 / 冲突关键词]
看看全球:
[影射/暗示主线事件]
```

### 节奏公式

**深度长文节奏：**
```
悬念标题(惊叹号) → 氛围铺垫 → 具体事件 → 技术细节 → 全球格局 → 评论/判断
```

**热搜快讯体：**
```
热词(惊叹号) → 引用来源解读 → 数据支撑 → 技术原文链接
```

### 关键术语词典

| 术语 | 使用场景 |
