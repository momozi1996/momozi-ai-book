# 05 — 创立逻辑与关键决策

## 1. 创立时机：AlphaGo掀起深度学习热潮（2015）
- 2014年「奇点临近」研讨会布局，瞄准深度学习上升期
- 2015年9月7日正式上线：与AlphaGo同年，移动互联网向AI社交平台迭代
- 2022年官方升级：「追求技术_chat_社交平台的第二曲线」

## 2. 三体模式：资讯社群 x 智库平台 x 产业基金
- 2015立品牌初年即提出三体模式——三层结构在一个品牌内协同
- 资讯内容（媒体平台）+ 认知层（专家智库）+ 经济层（产业基金）
- 这在国内外垂直媒体里是超前的战略构思

## 3. 从社群到媒体：2019-2022转型
- 由社群型产品转向媒体审视型平台
- 2019年战略升级：「南之元化」（合作伙伴称）
- 在2024/2025AI热潮中爆发：内容密度大幅提升
- 每日50+短讯+公号长文+AI对话产品三位一体

## 4. 人才布局策略
- 早期：杨静本人主导重量级报道
- 中期：引入元宇/Aeneas等编辑
- 2023-2024：团队化生产达到规模

## 5. 2024策略关键转变
- 增加每日快讯的数量和频率（每日50+条）
- 加入AI对话产品「新智AI」——从媒体进军产品
- 引入国际事件追踪（元宇HX等）

## 6. 差异化定位
- 与量子位（数据流+速报）不同：新智元叙事先于数据，冲突先于结论
- 与技术博主（个人IP型）不同：新智元是媒体机构而非个人IP
- 博主型媒体中拥有最丰富的历史感：「超智能时代」大规模活动自2016年起
